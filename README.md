# Animator Schedule Board / 애니메이터 스케줄 보드

제작 - 미야소

애니메이션 제작 스케줄을 CUT 단위로 관리할 수 있는 가벼운 단일 HTML 스케줄 보드입니다. 인터넷 연결, 서버, 설치 없이 `index.html` 파일 하나로 실행됩니다.

## 실행 방법

### 가장 쉬운 방법
1. `index.html` 파일을 더블클릭합니다.
2. 브라우저에서 스케줄 보드가 열립니다.
3. 바로 작품, 화수, CUT, TIME, LO, 체크, 원화/genga 상태를 입력합니다.

### Windows EXE 버전
1. `AnimatorScheduleBoard_Miyaso_Windows_x64.exe`를 실행합니다.
2. 앱이 기본 브라우저에서 자동으로 열립니다.
3. 이 EXE는 무거운 설치형 앱이 아니라, HTML 앱을 여는 가벼운 런처입니다.

## 기본 사용법

### 언어 변경
상단의 언어 선택에서 한국어 / 日本語 / English를 선택할 수 있습니다.

### 작품 / 화수 관리
- `프로젝트 추가` 버튼으로 새 작품을 추가합니다.
- `화수 추가` 버튼으로 현재 작품에 새 화수를 추가합니다.
- 작품명과 화수는 상단 입력칸에서만 수정합니다.
- 상단의 작품 선택 / 화수 선택 드롭다운으로 빠르게 이동할 수 있습니다.

### CUT 입력
- `CUT 추가` 버튼으로 컷을 추가합니다.
- `CUT+ / CUT-`로 여러 줄을 한 번에 추가하거나 줄일 수 있습니다.
- CUT 번호와 TIME은 표 안에서 직접 입력합니다.
- 방향키로 칸 이동이 가능합니다.

### LO / 체크 / 원화 상태
- LO 칸 클릭: 진행중 `▲` → 완료 `○` → 비움
- 원화/genga 칸 클릭: 진행중 `▲` → 완료 `○` → `二原` → 비움
- LO 또는 원화/genga 칸을 우클릭하면 상태를 직접 선택할 수 있습니다.
- 체크/戻し/Check 칸은 클릭으로 체크/해제합니다.

### 행 색상
- LO 완료: 초록색
- 진행중: 노란색
- 원화/genga 완료 또는 二原: 회색 + 반투명

### 접기 / 펼치기
- 작품 바의 `▼ / ▶` 버튼으로 작품 전체를 접거나 펼칠 수 있습니다.
- 화수 칸의 `▼ / ▶` 버튼으로 해당 화수만 접거나 펼칠 수 있습니다.

### 작품 순서 변경
- 작품 바의 `↕` 표시 또는 작품 바 영역을 드래그해서 작품 순서를 위아래로 바꿀 수 있습니다.
- 같은 작품에 포함된 화수들은 함께 이동합니다.

### 마감 계산
- 상단에서 마감일을 지정하면 오른쪽 하단에 D-Day가 표시됩니다.
- `D-Day`: 오늘 마감
- `D-N`: N일 남음
- `D+N`: 마감 N일 초과

### 데이터 저장 / 이동
- 데이터는 사용하는 브라우저에 자동 저장됩니다.
- 다른 컴퓨터로 옮기려면 `JSON 내보내기`를 누른 뒤, 새 컴퓨터에서 `JSON 가져오기`를 사용하세요.

## GitHub Pages로 공개하기

1. GitHub에서 새 Repository를 만듭니다.
2. 이 폴더의 `index.html`과 `README.md`를 업로드합니다.
3. Repository의 `Settings`로 이동합니다.
4. 왼쪽 메뉴에서 `Pages`를 선택합니다.
5. Source를 `Deploy from a branch`로 설정합니다.
6. Branch는 `main`, Folder는 `/root`를 선택합니다.
7. 저장 후 잠시 기다리면 GitHub Pages 주소가 생성됩니다.

## 주의사항

- 이 앱은 단일 HTML 파일 기반이라 가볍고 배포가 쉽습니다.
- 데이터는 각 사용자의 브라우저 저장소에 저장됩니다.
- GitHub Pages에 올려도 사용자들의 작업 데이터가 GitHub에 자동 업로드되지는 않습니다.
- 공유/백업은 JSON 내보내기 파일로 관리하세요.
