Animator Schedule Board

Animator Schedule Board は、アニメ制作における個人用のスケジュール管理ツールです。
主にレイアウト作業や原画作業を行うアニメーターが、自分の担当カット、進行状況、尺、締切を整理することを前提に作られています。

制作：みやそ

概要

Animator Schedule Board は、ブラウザで開くだけで使えるシンプルなスケジュールボードです。
作品ごと、話数ごとにカットを管理でき、LO、戻し、原画、二原などの進行状況を視覚的に確認できます。

インストールやサーバーは必要ありません。
index.html をブラウザで開くだけで使用できます。

GitHub Pages にアップロードすれば、URLを共有するだけで他の人も同じツールを開くことができます。
ただし、入力したスケジュールデータは各ユーザーのブラウザ内に保存されるため、他の人と自動で共有・同期されることはありません。

想定している使い方

このツールは、主に以下のような用途を想定しています。

レイアウト作業の進行管理
原画作業の進行管理
担当カットの一覧整理
カットごとの尺の確認
締切までの残り日数の確認
作品や話数ごとの作業量の把握
個人作業用の簡易スケジュール表

制作進行全体を管理するための本格的なチーム共有ツールではなく、
アニメーター個人が自分の作業状況を見やすく整理するための軽量なツールです。

主な機能
1. 作品ごとの管理

複数の作品を登録できます。
作品ごとにカット表が分かれるため、複数案件を同時に進めている場合でも整理しやすくなっています。

作品名は上部の入力欄から編集できます。
表の中の作品名セルは直接編集できない仕様になっており、誤操作を防ぐようにしています。

2. 話数ごとの管理

1つの作品の中に複数の話数を登録できます。
話数ごとにカット、尺、進行状況、締切を分けて管理できます。

上部の「作品選択」「話数選択」から、確認したい作品や話数へすぐに移動できます。

3. カット番号の入力

各行にカット番号を入力できます。
カット番号が空欄の行は、基本的に集計対象から外れます。

これにより、空の予備行を残しておいても、総カット数や進捗計算に影響しにくくなっています。

4. 尺の入力と自動集計

各カットに尺を入力できます。

入力例：

1+12
2+0
0+15

秒 + フレーム の形で入力することを想定しています。
入力された尺は話数ごとに自動集計され、各話数欄に総尺として表示されます。

尺を入力・変更すると、総尺も自動で更新されます。

5. LOのステータス管理

LO欄をクリックすると、状態を切り替えることができます。

切り替え順：

進行中 ▲ → 完了 ○ → 空欄

右クリックすると、メニューから直接状態を選択できます。

LO完了のカットは、行が緑色で表示されます。

6. 戻しの管理

戻し欄をクリックすると、チェックを付けることができます。
戻しが入っているカットは、行が黄色系で表示されます。

戻しの確認や修正が必要なカットを見つけやすくするための機能です。

7. 原画のステータス管理

原画欄をクリックすると、状態を切り替えることができます。

切り替え順：

進行中 ▲ → 完了 ○ → 二原 → 空欄

右クリックすると、メニューから直接状態を選択できます。

原画完了、または二原に設定されたカットは、グレー＋半透明で表示されます。
完了済み、または次工程に進んだカットを視覚的に分かりやすくするための表示です。

8. 二原表示

原画欄では、通常の完了表示とは別に 二原 を選択できます。

二原を選ぶと、原画欄に黒い 二原 表示が出ます。
行全体は完了状態と同じようにグレー＋半透明になります。

二原撒きや二原作業の把握に使えます。

9. 行の色分け

進行状況に応じて、行の色が自動で変わります。

LO完了：緑色
進行中：黄色
戻し：黄色
原画完了：グレー＋半透明
二原：グレー＋半透明

視覚的に進行状況を確認できるため、どのカットが残っているかを把握しやすくなっています。

10. 締切計算

話数ごとに締切日を設定できます。
締切日を入力すると、右側のパネルに締切までの日数が表示されます。

表示例：

D-Day：今日が締切
D-3：締切まであと3日
D+2：締切から2日経過

残りカット数も表示されるため、作業ペースの目安として使えます。

11. 作品・話数の折りたたみ

作品ごと、または話数ごとに表示を折りたたむことができます。

見なくてよい作品や話数を閉じておくことで、画面をすっきりさせることができます。

作品バーの ▼ / ▶ ボタンで作品全体を折りたたみできます。
話数欄の ▼ / ▶ ボタンで話数ごとに折りたたみできます。

折りたたんだ状態はブラウザに保存されます。

12. 作品順の並び替え

作品バーをドラッグすることで、作品の上下順を変更できます。

同じ作品に含まれる話数は、作品ごとまとめて移動します。
複数案件を優先度順に並べたい場合に便利です。

13. 方向キーでのセル移動

表の中では、方向キーでセルを移動できます。

↑：上のセルへ移動
↓：下のセルへ移動
←：左のセルへ移動
→：右のセルへ移動

CUTや尺を入力しながら、スプレッドシートのように操作できます。

14. JSONでのデータ保存・読み込み

入力したデータは、ブラウザ内に自動保存されます。
さらに、JSONファイルとして書き出し・読み込みができます。

JSON書き出し

現在のデータをJSONファイルとして保存できます。
バックアップや別PCへの移動に使えます。

JSON読み込み

保存しておいたJSONファイルを読み込むことで、以前のデータを復元できます。
